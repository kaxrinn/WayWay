
# WayWay
WayWay aplication web berbasis ai  yang menerapkan Artificial Intelligence berbasis NLP dan sistem rekomendasi untuk membantu wisatawan mendapatkan informasi, rekomendasi destinasi, serta itinerary perjalanan yang disesuaikan dengan preferensi pengguna.

## Tech Stack
- Laravel 12
- MySQL
- PHP 8+
- Tailwind CSS

## Fitur Utama
- Rekomendasi destinasi wisata berbasis preferensi pengguna
- Informasi destinasi wisata dalam satu kota
- Sistem itinerary perjalanan
